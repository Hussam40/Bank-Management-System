# 🏦 Hussam Bank Management System

A simple and interactive console-based Bank Management System built with Python. It allows users to manage bank clients, perform basic operations, and query the client list.

## 📌 Features
* **New Client Creation:** Automatically generates a unique 4-digit Account ID and assigns a status based on the deposited amount.
* **Client Portfolio Display:** Displays client information with a readable delay.
* **Search Functionality:** Search for a client by Account ID or Name.
* **Console UI:** Simple and clean terminal user interface with screen-clearing effects.

## 🚀 How to Run

### Prerequisites
* **Python 3.x** installed on your system.

### Steps
1. Clone or download the repository files.
2. Create or open the python file (e.g., `main.py`) and paste the code.
3. Run the script from the terminal:
